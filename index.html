responsive design

at the core of responsive web design is the ability to design a mobile-friendly website/App that adapts to different device sizes, including desktop, tablet, smartphone, etc. this enables users to browse a website/App within the constraints of their device. responsive design enables better user experience; and in a world where almost everyone uses a smartphone, it becomes smart and logical to design RESPONSIBLY. I am comfortable with using CSS media queries to set breakpoints for each screen size.

git & github

git is a version history for our code which allows us to save past versions of every file of our code so that whenever we need to we can view the code we had in the past, or, we can restor all of our code back to its past version. Git is a version control system that allows us manage and keep track of our source code history. it's goals include speed, data integrity, and support for distributed, non-linear workflows. GitHub on the other hand, is a cloud-based hosting service that enables us to manage Git repositories.
